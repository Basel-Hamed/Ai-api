# 🔒 প্রাইভেট ওয়েবসাইট API

এটি একটি **প্রাইভেট API** যা শুধুমাত্র আপনার কনফিগার করা ওয়েবসাইট থেকে তথ্য আনতে পারে।

## ✨ বৈশিষ্ট্য

- ✅ **শুধু আপনার দেওয়া সাইট থেকে তথ্য আনে**
- ✅ অন্য কোনো সাইট থেকে তথ্য আনা যায় না
- ✅ হেডলাইন, আর্টিকেল, সার্চ সুবিধা
- ✅ ইংরেজি থেকে বাংলা অনুবাদ
- ✅ ক্যাটাগরি অনুযায়ী ফিল্টার

## 📥 ইনস্টলেশন

```bash
# ১. ডাউনলোড
git clone https://github.com/yourusername/Private-Web-API
cd Private-Web-API

# ২. সেটআপ
chmod +x setup.sh
./setup.sh

# ৩. চালান
./run.sh

বিশেষ দ্রষ্টব্য: এই ফাইল এমটি ম্যানেজার দ্বারা এডিট করতে হবে অথবা আপনারা টার মার্কস দিয়ে এডিট করতে পারেন এখানে মেইন পি আই
ফাইল এ আপনাদের ওয়েবসাইট এর লিংক বসাবেন যেখান থেকে আপনারা ইনফরমেশন বা ডাটা আনতে চাচ্ছেন
